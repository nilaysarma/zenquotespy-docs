# Home

ZenquotesPy is a lightweight Python package that provides easy access to motivational and inspirational quotes from the ZenQuotes.io API. Whether you're building a chatbot, a productivity app, a daily quote generator, or just want a little inspiration in your terminal â€” this package makes it effortless. It is published as a Python package named `zenquotespy` on [PyPI](https://pypi.org/project/zenquotespy).

* Fetch a single random quote.
* Retrieve 50 quotes in bulk with one call.
* Get inspirational quote images.
* Clean and developer-friendly interface.

Inspirational quotes provided by <a href="https://zenquotes.io/" target="_blank">ZenQuotes API</a>
